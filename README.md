# cli-example
A simple CLI application

# Install in development mode to pick up changes
```
pip install -e .
```

# Use the CLI
The entry point script right now is defined as "mycli". It has one command implemented:
```
$ mycli hello Developer!
  ________________
| Hello Developer! |
  ================
                     \
                      \

                       |o_o |

                     (|     | )
                    /'\_   _/`\
                    \___)=(___/

```
